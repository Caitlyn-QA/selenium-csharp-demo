# Selenium Csharp Demo

A Selenium WebDriver project built with C# and NUnit against the Practice Software Testing application.

## Purpose

This project recreates selected UI automation scenarios from my Playwright portfolio to compare different automation frameworks and strengthen understanding of browser automation fundamentals.

## Technology Stack

- C#
- .NET 10
- Selenium WebDriver
- NUnit
- ChromeDriver

## Current Coverage

### Product Search
- Search for products using the application search functionality.
- Verify that search results match the entered search criteria.

### Product Navigation
- Open a product from the product overview page.
- Verify navigation to the correct product detail page.

## Key Concepts Demonstrated
Page Object Model (POM)
Explicit waits using WebDriverWait
UI synchronization handling
Selenium WebDriver fundamentals
NUnit test structure

## Notes

This project was created as a learning exercise to compare Selenium WebDriver and Playwright implementations of similar test scenarios.
While Playwright provides built-in auto-waiting and modern testing features, Selenium offers a more explicit view of browser automation and synchronization, helping deepen understanding of automation fundamentals.
